# The Code Test
The code test for search and calculate cart price based on two json files.

## :dog: Setup
This project needs Java `8` and Scala `2.12.3`
[Install Java](https://java.com/en/download/help/index_installing.xml)

```bash
# Install Scala
$ brew install scala
$ brew install sbt

# Compile project
$./sbt clean compile
```
